# plv.io

一个简约的个人技术博客。

## 博客简介

plv.io 是基于 Nuxt 3 构建的个人技术博客，采用工业风设计。主要用于记录技术学习笔记、项目经验和思考总结。

## 功能

- 文章发布与分类浏览
- 项目展示与 Demo 链接
- 留言板互动
- 暗色/亮色模式切换
- 响应式布局，支持移动端

## 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | Nuxt 3 |
| UI 层 | Vue 3 + UnoCSS |
| 内容管理 | Nuxt Content |
| 图标 | Iconify (Carbon + MDI) |
| 部署 | Vercel / Netlify |
